## Technogologies

* CSS3
* React
* React-Redux
* React Router
* Styled Components
* Pure Functional Components

## Installation

```
$ mkdir ReactUI
$ cd ReactUI
$ git clone https://github.com/brianeddow/react-ui.git
$ npm install
$ npm build
$ npm start
```
